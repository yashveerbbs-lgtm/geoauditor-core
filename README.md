GeoAuditor: Enterprise PropTech Solution

GeoAuditor is a full-stack geospatial platform designed to simplify land surveying, plot optimization, and data management. It combines high-performance geometric math engines with secure cloud storage to provide real-time boundary visualization and analysis.

Live Demo

Access the platform here: https://geoauditor-app.onrender.com

🛠 Tech Stack
Frontend: HTML5, CSS3 (Glass-morphism UI), JavaScript (Leaflet.js for interactive mapping)

Backend: Python FastAPI (RESTful API architecture)

Database: SQLite (Secure Cloud Vault)

Deployment: Render (Cloud Backend & Static Site Hosting)

✨ Core Features
Dynamic Map Editor: Real-time marker placement and instant polygon rendering.

Geometric Optimization: Backend math engine that resolves overlapping/distorted coordinates to create clean, optimized land boundaries.

Cloud Vault: Persistent storage allowing users to save, name, and retrieve property plots from a secure database.

Responsive Design: Enterprise dashboard UI compatible with mobile and desktop devices.

💡 How to Use
Define Area: Click on the map to drop coordinates for your property boundary.

Optimize: Click the "Optimize" button to send data to the math engine for geometric cleanup.

Save: Enter a plot name to store the optimized boundary in the cloud vault.

Retrieve: Access the "Saved Plots" sidebar to instantly load and zoom to any previously saved property.
